# Bushmills
Bushmills is a attendance management system build by TypeScript/React/nodejs/Express

## Pages

- Login
- Timecard
- List (only admin)
- Detail (only admin)

## Functions

- Timecard
  - set work in
  - set work out
  - set break in
  - set break out
- List
  - add user info
  - edit user info
- Detail
  - insert remark
  - edit work in
  - edit work out
  - calc break hours
  - calc work hours
  - edit break time
