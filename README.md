# Bushmills

Bushmills is a attendance management system.
It has two mainly functions.

1. Timecard
2. Attendance list on the month.

build by TypeScript/React/nodejs/Express.

## Pages

- Login
- Timecard
- Detail

## Functions

- Timecard
  - set work in
  - set work out
  - set break in
  - set break out
- Detail
  - insert remark
  - edit work in
  - edit work out
  - calc break hours
  - calc work hours
  - edit break time
